# 🚀 NovaPkg
A high-performance, branded wrapper for the Arch Linux `pacman` package manager. 

NovaPkg simplifies package management while adding a custom aesthetic experience to your terminal.

## 🛠 Installation

To install NovaPkg on your Arch system, run the following commands:

```bash
# Download the script
curl -LO [https://raw.githubusercontent.com/dexomax/novapkg/main/novapkg](https://raw.githubusercontent.com/dexomax/novapkg/main/novapkg)

# Make it executable
chmod +x novapkg

# Move it to your path
sudo mv novapkg /usr/local/bin/
